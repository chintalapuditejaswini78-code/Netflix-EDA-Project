# Netflix Data Analysis Project

## Objective

To perform Data Cleaning and Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python.

## Tools Used

* Python
* Pandas
* Matplotlib
* VS Code

## Dataset

Netflix Movies and TV Shows Dataset

## Data Cleaning

* Checked missing values
* Filled missing values
* Converted date columns
* Checked duplicates
* Saved cleaned dataset

## Exploratory Data Analysis (EDA)

* Movies vs TV Shows
* Release Year Distribution
* Top 10 Countries by Content
* Content Added Per Year

## Key Insights

1. Netflix has more Movies than TV Shows.
2. Most content was released after 2015.
3. The United States contributes the highest amount of content.
4. Netflix content increased rapidly in recent years.
5. The dataset was successfully cleaned with no missing values.
